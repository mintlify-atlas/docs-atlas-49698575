## Get Lyrics by Song ID

Retrieve the lyrics for a specific song using its unique identifier.

### Endpoint

```
GET /api/v1/SongsData/GetLyricsById
```

### Authentication

<Warning>
  This endpoint requires authentication. Include a valid bearer token in the Authorization header.
</Warning>

### Query Parameters

<ParamField query="id" type="string" required>
  The unique identifier of the song whose lyrics are to be retrieved.
</ParamField>

### Response

<ResponseField name="data" type="string">
  The lyrics text for the requested song. Lyrics are returned as plain text with line breaks.
</ResponseField>

### Error Responses

<ResponseField name="status" type="integer">
  HTTP status code
</ResponseField>

<ResponseField name="message" type="string">
  Error description
</ResponseField>

#### Status Codes

- `200` - Success
- `401` - Unauthorized (missing or invalid authentication)
- `404` - Not Found (lyrics not found for the specified song)

### Examples

<CodeGroup>

```bash Get Lyrics
curl -X GET "https://api.swar.com/api/v1/SongsData/GetLyricsById?id=abc123" \
  -H "Authorization: Bearer YOUR_TOKEN"
```

</CodeGroup>

### Usage Examples

<CodeGroup>

```bash Lyrics Only
curl -X GET "https://api.swar.com/api/v1/SongsData/GetLyricsById?id=abc123" \
  -H "Authorization: Bearer YOUR_TOKEN"
```

```bash Song with Lyrics

```

</CodeGroup>

### Notes

- Not all songs have lyrics available in the source database
- If lyrics are not available, a 404 error is returned
- Lyrics are returned as plain text with preserved line breaks and formatting
- This endpoint is useful when you only need lyrics without fetching full song metadata
- To check if a song has lyrics before making this request, use the [Get Song by ID](/api/song-service/songs) endpoint and check the `has_lyrics` field
